About Deep-Vis

This application is based on the paper "Plug-and-Play Interactive Deep Network Visualization" of G. Strezoski & al.

It uses Keras, a high-level neural networks API written in Python and running on top of Tensorflow, and Keras-Vis, a neural network visualisation toolkit for keras, in order to process the visualizations.
